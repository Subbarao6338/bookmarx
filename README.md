# NECS Bookmarks

A streamlined bookmark management system, designed for professional efficiency and organized link access.

## 🚀 Features

*   **Professional UI**: Clean, Material-inspired cards with smooth interactive states.
*   **Performance**: Optimized React components for maximum speed.
*   **PWA Support**: Fully installable with offline capabilities via service workers.
*   **Customization**: Material Expressive color palettes and theme modes (Light/Dark/System).
*   **Privacy First**: Local-first processing and storage.

## 💻 Tech Stack

*   **Frontend**: React 18, Vite, CSS3 (Material Expressive Design).
*   **Data**: JSON-based local storage.

## 💻 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Built for those who value speed and simplicity.
